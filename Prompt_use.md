# AI Prompts Used

## Purpose

AI was used only as a research assistant to organize information, improve documentation, and structure the final deliverables. All company details were manually reviewed using publicly available sources before being added to the spreadsheet.

---

## Prompts Used

### Prompt 1

> Find manufacturing companies in Hyderabad that operate in aerospace, defence, precision engineering, medical devices, industrial electronics, and specialty manufacturing.

### Prompt 2

> Summarize the products manufactured by this company and identify whether it qualifies as a product-based manufacturing company.

### Prompt 3

> Help organize company information into a spreadsheet with the required columns for the assignment.

### Prompt 4

> Draft a professional research methodology document explaining the research process, qualification criteria, sources used, and key learnings.

### Prompt 5

> Create a professional README file for a GitHub repository containing this assignment.

### Prompt 6

> Suggest personalization hooks based on publicly available company information.

## Manual Verification

After using AI for organization and documentation, I manually reviewed company websites and other public sources to verify company information before finalizing the spreadsheet.
